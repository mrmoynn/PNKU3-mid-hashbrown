# AGENT.md — Reusable Prompts

---

## catalog — Refresh the Tool Catalog

```
Browse the entire repository structure. For every tool, script, or solver you find,
update #Docs/catalog.md with a brief description of what it does, its key files,
input/output formats, and any notable algorithm or design choices.

Organize entries by category (matching the directory structure). Keep it concise —
one paragraph per tool plus a small file-role table where helpful. End with a tree
of the repo layout.

Include only what the code actually does; do not invent features.
```
